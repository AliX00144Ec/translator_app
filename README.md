A simple web application built with **Python Flask** to translate sentences from **Italian to Arabic** using the `googletrans` library.

---

## **Features**

- Simple and clean interface with **HTML + CSS + Bootstrap**  
- Instant translation from Italian to Arabic  
- Can be run **locally** or **shared online** using Ngrok  
- Ready to deploy on hosting platforms like **Render** or **Railway**

---

- Python 3.10 or higher  
- Python libraries:
